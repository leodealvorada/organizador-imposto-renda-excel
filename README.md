# Organizador de Declaração de Imposto de Renda

Este projeto contém uma planilha em Excel desenvolvida no curso da DIO, com o objetivo de facilitar a organização das informações necessárias para a declaração anual do Imposto de Renda.

## Funcionalidades
- Registro de rendimentos
- Controle de despesas dedutíveis
- Organização de bens e direitos
- Resumo automático para conferência

## Como usar
1. Baixe o arquivo `Organizador_IR.xlsx`.
2. Abra no Excel ou similar.
3. Preencha suas informações nas abas correspondentes.
4. Utilize os resumos para validar os dados antes da declaração.

## Tecnologias
- Microsoft Excel
- Fórmulas e tabelas dinâmicas

## Créditos
Projeto desenvolvido durante o curso **Criando um Organizador de Declaração de Imposto de Renda** da [DIO](https://web.dio.me).
